《怪獸異世界》遊戲說明書

一個以深度客製化與 AI 互動為核心的怪獸養成遊戲

遊戲總覽

《怪獸異世界》是一款深度策略型怪獸養成遊戲，其核心魅力在於無限的可能性。遊戲捨棄了傳統固定的怪獸設計，讓玩家成為真正的「造物主」。透過組合來自不同元素、不同稀有度的「DNA碎片」，玩家可以創造出外觀、數值、技能組合都獨一無二的怪獸夥伴。

遊戲不僅注重策略性的戰鬥與養成，更創新地引入了 AI 互動系統。每一隻怪獸都擁有由 AI 生成的獨特背景故事、養成評價、以及基於其「個性」和與玩家「牽絆值」的動態聊天回應。這不僅僅是養成，更是一場與您親手創造的數位生命建立情感連結的旅程。

獨特之處

  • 無限創造： 透過數百種 DNA 碎片的組合，打造出理論上無限種可能的怪獸。您的想像力是唯一的限制。
  • 深度策略： 屬性相剋、技能搭配、個性影響，每一場戰鬥都是對玩家智慧的考驗。沒有最強的怪獸，只有最強的訓練師。
  • 情感連結： 創新的 AI 聊天與互動系統，讓每隻怪獸都成為有血有肉的夥伴，牠們會記住您的互動，並展現出不同的態度。
  • 公平競技： 遊戲內不存在付費變強的機制，所有成長皆來自於遊戲內的努力與策略，提供一個純粹的競技與養成環境。

核心玩法詳解

🧬 DNA 合成系統

這是您創造生命的起點。遊戲的核心是收集並組合DNA碎片。

  • 來源： 您可以透過每日免費抽卡、完成「冒險島」的探索、或參與未來開放的活動來獲得DNA碎片。
  • 組合規則： 將5個DNA碎片放入組合槽，點擊「怪獸合成」即可。
  • 數值與屬性： 新怪獸的基礎HP、MP、攻擊、防禦等數值，是5個DNA素材數值的總和。而怪獸的主屬性，由數量最多的元素決定（例如：3個火DNA + 2個水DNA = 火屬性怪獸）。
  • 稀有度： 怪獸的最終稀有度，取決於您使用的5個DNA中，稀有度最高的那一個。使用「傳奇」DNA是誕生傳說怪獸的關鍵！

⚔️ 全自動戰鬥系統

戰鬥是檢驗您成果的最佳方式，但《怪獸異世界》的戰鬥更側重於戰前策略而非戰中操作。

  • 戰前準備： 在「怪獸農場」選擇您要「出戰」的怪獸。牠將代表您參與所有挑戰。
  • 自動模擬： 一旦發起挑戰，後端伺服器將根據雙方怪獸的數值、屬性克制、技能組、AI個性、甚至亂數，完整模擬一場激烈的戰鬥。
  • 戰後分析： 您會收到一份由AI生成的、生動的文字戰報。分析戰報，找出優劣勢，是您優化怪獸、調整策略的關鍵。

🏞️ 修煉與成長

修煉是怪獸變強的根本途徑，也是一個充滿隨機性的掛機系統。

  • 派遣修煉： 在農場中選擇怪獸，並為牠選擇一個修煉地點。不同的地點會對特定屬性的怪獸有額外的成長加成。
  • 能力提升： 修煉結束後，怪獸的技能會獲得經驗值並升級。技能在達到5級和10級時會解鎖強大的「里程碑」效果！
  • 意外收穫： 修煉過程中，怪獸可能會領悟全新的技能，或拾獲稀有的DNA碎片與道具，每一次修煉都充滿驚喜。

💖 AI互動與情感連結

我們希望每隻怪獸都是獨一無二的夥伴，而不只是冰冷的數據。

  • 專屬故事： 每隻怪獸誕生時，AI都會為牠生成獨特的背景故事和養成評價，您可以在怪獸的詳細資訊中查看。
  • 動態對話： 在怪獸的「對話」頁籤，您可以和牠聊天。牠的回應會基於牠的「個性」（如勇敢、膽小）和你們之間的「牽絆值」而改變。
  • 提升牽絆： 透過聊天、摸摸頭等互動，可以提升牽絆值。高牽絆的怪獸不僅更親密，甚至可能在您請求牠開關戰鬥技能時，更願意聽從您的建議！

技術架構與使用工具

後端 (Backend)

  • 語言/框架: Python / Flask
  • 伺服器: Gunicorn on Render.com
  • 資料庫: Firebase Firestore (NoSQL)
  • 身份驗證: Firebase Authentication
  • AI模型: DeepSeek API

前端 (Frontend)

  • 核心: JavaScript (ES6), HTML5, CSS3 (無前端框架)
  • 部署: GitHub Pages
  • 主要工具: Fetch API 進行非同步請求

已安裝與使用的程式

  • 後端套件: Flask, firebase-admin, Flask-CORS, gunicorn, pytest, noise, PyJWT。
  • 版本控制: Git / GitHub。
  • 開發工具: 程式碼編輯器 (如 VS Code), 瀏覽器開發者工具。

發展藍圖與規劃

我們對《怪獸異世界》的未來充滿期待，並已規劃了清晰的發展路線，以持續為玩家帶來新鮮感與深度體驗。

短期計畫 (1-3個月)

目標： 完善現有核心系統，豐富遊戲內容。

  • 武道大會實裝： 完成PVP天梯系統，引入段位、賽季與獎勵機制。
  • 醫療站功能擴充： 增加基因改造、DNA摘除等深度客製化功能。
  • 訓練場開放： 提供一個無獎勵的PVP切磋場所，方便玩家測試陣容。
  • 內容填充： 大量增加新的DNA碎片、技能、冒險事件與BOSS。

中期計畫 (3-9個月)

目標： 引入大型新系統，強化社群互動。

  • 家園系統： 玩家將擁有可自訂裝飾的個人空間，用於展示心愛的怪獸。
  • 交易所系統： 開放玩家間的DNA與道具交易，建立遊戲內經濟。
  • 公會系統 (初版)： 引入基礎的公會創建、成員管理與聊天頻道。

長期願景 (9個月以上)

目標： 打造可持續發展的遊戲生態，探索更多可能性。

  • 主線劇情任務： 推出帶有豐富劇情的任務線，引導玩家探索異世界的秘密。
  • 多人合作冒險 (PVE)： 設計需要多名玩家組隊才能挑戰的巨大BOSS或特殊副本。
  • 怪獸情感族譜： 建立更深度的怪獸關係系統，例如怪獸間的好感度、組合技等。

常見問題 Q&A

Q: 這款遊戲需要付費嗎？

A: 完全不需要。本遊戲的核心理念是公平競技與養成樂趣，沒有任何付費變強的機制。所有內容都需要透過遊戲內的努力來獲得。

Q: 我該如何獲得更強的怪獸？

A: 變強的途徑有很多！

  • 稀有DNA： 透過「冒險島」或活動獲得高稀有度的DNA，這是合成強力怪獸的基礎。
  • 修煉： 持續派遣怪獸去修煉，可以提升牠們的基礎數值和技能等級。
  • 策略： 深入研究屬性克制與技能搭配，用戰術智慧以弱勝強。
  • 玩家稱號： 解鎖並裝備強力稱號，其屬性加成會對你所有怪獸生效！

Q: 我卡關了或發現了BUG，該怎麼辦？

A: 非常歡迎您回報！由於遊戲仍在開發中，您的回饋至關重要。您可以透過信箱系統中的「聯繫客服」功能，或直接加入我們的官方LINE社群來聯繫開發者。
