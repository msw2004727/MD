《怪獸異世界》遊戲白皮書

一個以深度客製化與 AI 互動為核心的怪獸養成遊戲

遊戲總覽

《怪獸異世界》是一款深度策略型怪獸養成遊戲，其核心魅力在於無限的可能性。遊戲捨棄了傳統固定的怪獸設計，讓玩家成為真正的「造物主」。透過組合來自不同元素、不同稀有度的「DNA碎片」，玩家可以創造出外觀、數值、技能組合都獨一無二的怪獸夥伴。

遊戲不僅注重策略性的戰鬥與養成，更創新地引入了 AI 互動系統。每一隻怪獸都擁有由 AI 生成的獨特背景故事、評價、以及基於其「個性」和與玩家「牽絆值」的動態聊天回應。這不僅僅是養成，更是一場與您親手創造的數位生命建立情感連結的旅程。

核心理念

  • 無限創造： 透過數百種 DNA 碎片的組合，打造出理論上無限種可能的怪獸。
  • 深度策略： 屬性相剋、技能搭配、個性影響，每一場戰鬥都是對玩家智慧的考驗。
  • 情感連結： 創新的 AI 聊天與互動系統，讓每隻怪獸都成為有血有肉的夥伴。
  • 公平競技： 遊戲內不存在付費變強的機制，所有成長皆來自於遊戲內的努力與策略。

核心玩法 (已實裝)

🧬 DNA 合成系統

遊戲的起點與核心。玩家需將 5 個 DNA 碎片放入組合槽進行合成。最終誕生的怪獸，其基礎數值由 5 個 DNA 的總和決定，主屬性由數量最多的元素決定，而稀有度則取決於放入的最高稀有度 DNA。這一設計鼓勵玩家不斷實驗，尋找黃金組合公式。

⚔️ 深度戰鬥系統

戰鬥採用回合制。系統會綜合考量怪獸的數值、屬性相剋、技能效果、狀態異常以及由「個性」決定的 AI 行為模式，自動模擬出一場完整的戰鬥。玩家的樂趣在於戰前的策略佈局與戰後的戰報分析，而非手動操作。

🏝️ PVE 冒險島

一個充滿隨機性的 PVE 挑戰內容。玩家需組建 1-3 隻怪獸的遠征隊，探索充滿未知事件與強大 BOSS 的樓層式地圖。每一次的選擇都可能帶來獎勵或懲罰，成功擊敗樓層 BOSS 後可獲得金幣與稀有 DNA 等豐厚回報。

💖 AI 互動與情感連結

每隻怪獸都有由 AI 生成的獨特背景故事與養成評價。更重要的是，玩家可以透過專屬的聊天頁籤與怪獸進行 AI 對話。怪獸的回應會根據其「個性」（如勇敢、膽小、傲慢）以及與玩家的「牽絆值」動態變化，從冷漠到親密，建立獨一無二的情感連結。

🏆 社交與排行榜系統

遊戲內建完整的好友系統，玩家可以搜尋、新增好友，並透過信箱系統進行交流。信箱不僅用於玩家間通訊，也負責接收系統公告、好友請求及每日的冠軍獎勵。玩家可以挑戰「怪獸排行榜」上的對手，爭奪「冠軍殿堂」的榮譽席位。

技術架構

後端 (Backend)

  • 框架: Flask (Python) - 輕量、快速，適合快速迭代開發。
  • 部署: Gunicorn on Render.com - 成熟的 WSGI 伺服器，能穩定地在 PaaS 平台上運行。
  • 資料庫: Firebase Firestore - NoSQL、雲端託管的資料庫，擴展性強，能靈活應對遊戲資料結構的變化。
  • AI 模型: DeepSeek API - 用於生成怪獸的背景故事、評價，以及驅動與怪獸的 AI 聊天互動。

前端 (Frontend)

  • 核心: Vanilla JavaScript (ES6) - 無框架，提供最大的靈活性與性能，並透過模組化檔案結構進行管理。
  • 樣式: Pure CSS - 手工編寫的 CSS，實現了深色/淺色主題切換，並針對遊戲風格進行了客製化。
  • 溝通: Fetch API - 使用瀏覽器原生的 Fetch API 與後端進行非同步數據交換。
  • 認證: Firebase Authentication - 處理玩家的註冊、登入與身份驗證。

發展藍圖

基於目前的穩定架構，我們為《怪獸異世界》規劃了清晰的短、中、長期發展路線，以持續豐富遊戲內容與玩家體驗。

短期計畫 (1-3個月)

目標：完善現有功能，豐富核心內容。

  • 實裝醫療站： 開放基因改造、DNA 摘除、藥劑注射等功能。
  • 開放訓練場： 建立一個可以讓怪獸之間進行無獎勵切磋的場所。
  • 內容擴充： 大量增加新的 DNA 碎片、技能、冒險事件與 BOSS，提升遊戲可玩性。
  • UI/UX 優化： 持續根據玩家回饋，優化操作流程與介面顯示。

中期計畫 (3-9個月)

目標：引入新的核心系統，增強玩家間的互動。

  • PVP 武道大會： 實裝非同步的玩家對戰系統，建立獨立的PVP天梯與賽季獎勵。
  • 家園/秘密基地系統： 讓玩家擁有一個可以自由佈置、展示怪獸的個人空間。
  • 交易所商人： 開放玩家間的物品（如稀有DNA）交易系統，建立健康的遊戲內經濟。
  • 公會系統初版： 引入基礎的公會創建、成員管理與公會頻道功能。

長期願景 (9個月以上)

目標：打造一個可持續發展的、充滿活力的遊戲生態。

  • 主線劇情與任務： 推出帶有豐富劇情的任務線，引導玩家探索異世界的秘密。
  • 多人合作冒險： 開發需要多名玩家組隊才能挑戰的巨大 BOSS 或特殊副本。
  • 定期線上活動： 舉辦季節性活動、節日慶典，推出限定的怪獸與獎勵。
  • 怪獸情感族譜： 建立更深度的怪獸關係系統，例如怪獸間的好感度、組合技等。
  • 跨平台可能性： 評估並探索將遊戲移植到行動裝置等其他平台的可能性。
