專案結構圖 (Project Structure)
---------------------------------

/
|
|-- backend/                       # 後端應用程式主要資料夾
|   |
|   |-- __init__.py                # 將 backend 資料夾標記為一個 Python 套件
|   |
|   |-- main.py                    # Flask 應用的主入口檔案，初始化 App 和 Blueprint
|   |
|   |-- admin_routes.py            # 定義後台管理介面 (Admin Panel) 的所有網頁路由 (Routes)
|   |
|   |-- auth_routes.py             # 定義使用者身份驗證相關的路由，如登入、註冊、登出
|   |
|   |-- game_routes.py             # 定義核心遊戲邏輯的 API 路由
|   |
|   |-- models.py                  # 定義資料庫模型 (Schema)，使用 SQLAlchemy 與資料庫互動
|   |
|   |-- database.py                # 處理資料庫連線設定與 Session 管理
|   |
|   |-- config.py                  # 存放應用程式的設定，例如資料庫路徑、密鑰等
|   |
|   |-- services/                  # 服務層，存放主要的商業邏輯，讓路由檔案保持簡潔
|   |   |-- __init__.py
|   |   |-- user_service.py        # 處理使用者相關的邏輯 (新增、查詢、更新、刪除)
|   |   |-- character_service.py   # 處理遊戲角色相關的邏輯
|   |   |-- item_service.py        # 處理遊戲物品相關的邏輯
|   |   |-- quest_service.py       # 處理遊戲任務相關的邏輯
|   |   |-- transaction_service.py # 處理交易或內購相關的邏輯
|   |
|   |-- analytics/                 # 數據分析相關模組
|   |   |-- __init__.py
|   |   |-- analytics_services.py  # 提供數據分析相關的服務，如計算 DAU, MAU 等指標
|   |
|   `-- utils/                     # 工具函式模組
|       |-- __init__.py
|       `-- helpers.py             # 存放可重用的輔助函式
|
|-- frontend/                      # 前端檔案主要資料夾
|   |
|   |-- templates/                 # 存放所有由 Flask 渲染的 HTML 模板
|   |   |-- index.html             # 遊戲主頁面
|   |   |-- login.html             # 使用者登入頁面
|   |   |-- register.html          # 使用者註冊頁面
|   |   |
|   |   `-- admin/                 # 後台管理介面的相關模板
|   |       |-- dashboard.html     # 後台儀表板
|   |       |-- users.html         # 使用者管理頁面
|   |       |-- characters.html    # 角色管理頁面
|   |       |-- items.html         # 物品管理頁面
|   |       |-- quests.html        # 任務管理頁面
|   |       |-- transactions.html  # 交易管理頁面
|   |       `-- edit_user.html     # 新增或編輯使用者的表單頁面
|   |
|   `-- static/                    # 存放靜態檔案，如 CSS, JavaScript, 圖片等
|       |
|       |-- css/                   # 存放 CSS 樣式表
|       |   `-- style.css          # 主要的樣式檔案
|       |
|       `-- js/                    # 存放 JavaScript 檔案
|           |-- main.js            # 主要的遊戲前端邏輯
|           `-- api.js             # 專門用來與後端 API 溝通的函式庫
|
|-- tests/                         # 自動化測試資料夾
|   |-- __init__.py
|   `-- test_api.py                # 針對 API 進行測試的腳本
|
|-- .gitignore                     # Git 版本控制的忽略清單，避免將不必要的檔案上傳
|
|-- README.md                      # 專案的說明檔案，介紹如何安裝、設定與執行
|
`-- requirements.txt               # 記錄專案所有 Python 相依套件與版本，方便部署
